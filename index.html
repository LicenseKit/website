The LicenseKit SDK helps you protect your software with commercial licenses. It supports all major Apple platforms (iOS, iPadOS, macOS, tvOS and watchOS) and can validate licenses on-device and over remote web requests.

Rich Capabilities

LicenseKit lets you set up licenses with code, read licenses from files, validate licenses over remote web requests, etc. Licenses can specify and validate expiration date, platform (iOS, macOS, etc.), bundle ID, tier, and much more.

Affordable Pricing

LicenseKit is free to start and affordable to scale for indie developers, startups, businesses and enterprises.
